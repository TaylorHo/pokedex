Como Jogar?

1. Procure um Pokemon pelo nome ou pela lista (acessando a aba "Todos os Pokemons").


2. Selecione o Pokemon que deseja capturar, e clique em "Capturar"


3. Clique em "Rolar o Dado". Se você tirar um valor maior que 3, capturou o Pokemon com sucesso!! Se não, tente novamente!


4. Acesse a página inicial para ver quais Pokemons você já capturou!
